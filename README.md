# Guild AgentDesk Dogfood

This repo is a clean external dogfood target for Guild AgentDesk alpha.

The goal is to prove the public install path:

- bootstrap GitHub issue intake
- turn an `agent:ready` issue into a mandate
- claim the mandate
- attach proof
- verify the Agent Work Contract
- export replay
- open a PR with the proof artifacts

## Dogfood Runs

- Issue #1 / PR #2 proved the original public alpha loop.
- Issue #3 proves the TypeScript-first `guild-agentdesk` loop.
